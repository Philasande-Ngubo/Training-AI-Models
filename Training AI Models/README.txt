This Program contains graphical user interface 
Its tested in Linux and Windows

On Linux terminal open the folder as directory on the terminal:

-> to run the BST app run this command   "make GenericsKbBSTApp"
-> to run the Array app run this command "make GenericsKbArrayApp"

On Windows open the source code on the src folder using the IDE of your choice.
you can run  GenericsKbBSTApp.java or GenericsKbArrayApp.java for interaction.
